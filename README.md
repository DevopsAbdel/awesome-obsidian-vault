# Awesome Obsidian Vault

This repository can host multiple Obsidian vaults.

## Add A New Vault

1. Add your vault as a top-level folder in this repository.
2. Make sure the folder contains a `.obsidian/` directory.
3. Commit and push your changes.

The README vault index is auto-generated by `scripts/update-readme-vaults.sh` and updated automatically in CI.
